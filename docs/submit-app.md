# Submit an App

To request an app for devices:

1. Open an Issue using the app submission template.
2. Provide the exact app name.
3. Provide the Android package name.
4. Provide the version you need.
5. Provide a direct APK download link.
6. Provide the official source website.
7. Explain what the app is used for.

Notes:

- Approval is manual.
- Submitting an Issue does not guarantee inclusion.
- Only reviewed apps are added to the device registry.
